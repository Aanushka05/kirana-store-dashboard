# kirana-store-dashboard
Interactive Kirana Store Analytics Dashboard built using Power BI and Excel to analyze sales, profit, customer behavior, payment methods, and category-wise performance.

## Project Overview

This Power BI dashboard provides insights into a Kirana Store's sales performance, profit trends, customer purchases, and payment methods. The dashboard helps identify top-selling products, profitable categories, and customer buying patterns.

## Tools Used
- Power BI
- Excel
- Data Visualization
- Business Analytics

## Features
- Profit Analysis Dashboard
- Category-wise Sales Analysis
- Customer Purchase Analysis
- Payment Method Distribution
- Monthly Profit Tracking
- Interactive Filters and Slicers
